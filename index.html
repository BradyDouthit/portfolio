Real time chat!

Back-end and middleware done with MongoDB, Passport.io, BCrypt, Express, and Express Session. Front-end done with React and Socket.io. Made by myself, Charles Mercer, Ashlie Johnson, and Cory Walker.

Indecision

Simple app I was inspired to make when I couldn't decide where to eat.

Developed with the Foursquare API and React, this app allows you to put in your zip code and get back restaurants at that zip code. I then process the results and spit out five restaurants to choose from. This project is still in development, however I am excited about sharing it.

React Clicky Game

Minecraft themed logic game built with React

Built with React, this is a memory game in which there are 10 "blocks" which are randomized or shuffled after each click. The goal is to only click each block one time. If you can click all of them once, you win!

Crystal Puzzle Game

Simple JavaScript puzzle game

Built with pure HTML, CSS, JavaScript, and jQuery. This is a game in which there are 4 (or 5 if you are good enough) crystals each with a hidden random value. There is a target number (also randomized) which you can see on the screen. That number is the goal. If you go over the target number you will lose.
